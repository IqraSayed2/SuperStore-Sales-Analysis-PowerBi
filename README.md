# Superstore Sales Dashboard

## Objective

To create a simple dashboard that shows sales data from a superstore, helping beginners understand trends in sales, profits, and customer behavior.

## Tools

- Power BI: For making charts and dashboards.
- CSV file: For the data.

## Dataset

- File: SuperStore_Sales_Dataset.csv
- Contains: Orders, products, customers, sales amounts, profits, etc.

## Methodology

1. Load the CSV data into Power BI.
2. Clean and organize the data.
3. Create charts and graphs to show different views of the data.
4. Add a forecast for future sales.

## Key Insights

- Total Sales: $1.6 million
- Top Category: Office Supplies
- Top State: California
- Most Used Payment: Cash on Delivery (43%)
- Consumer segment buys the most (48%)
